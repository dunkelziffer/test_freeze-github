# test freeze

This is a companion repo for https://github.com/dunkelziffer/frozen_rails.

Branches:
- `main` a base branch with a new empty Rails project and some initial setup preparations
- `frozen-<name>` (e.g. `frozen-db`): Shows the current output of generator `frozen:<name>`
- `frozen-<name>-target` (e.g. `frozen-db-target`): Shows the desired output of generator `frozen:<name>`

## Contributing

For contributions, please open PRs against the correct `-target` branch:
- Bugfixes are always welcome
- Please open a discussion first before investing time into additions and behavior changes
